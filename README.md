# 🌦️ Nimbus – Modern Weather App

Nimbus is a sleek and responsive weather web application that provides real-time weather updates with a clean and modern UI. It allows users to search for any city worldwide or use their current location to get accurate weather insights instantly.

---

## ✨ Features

- 🔍 Search weather by city name  
- 📍 Get weather using current location (Geolocation API)  
- 🌡️ Real-time temperature, humidity, and wind speed  
- ☁️ Dynamic weather icons from OpenWeather  
- ⚡ Smooth animations and responsive design  
- 🚫 Error handling for invalid locations and network issues  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Glassmorphism UI)  
- JavaScript (JS)  
- OpenWeather API  

---

## 🚀 How It Works

1. User enters a city name or uses the location button  
2. The app fetches weather data from OpenWeather API  
3. Data is processed and displayed dynamically on the UI  
4. Error handling ensures smooth user experience
